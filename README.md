# Student-Exam-Regression-From-Scratch
Performed linear regression from scratch by using python to code a gradient descent algorithm then adjusting learning rate and epochs to reach a optimal regression line. 
Only used pandas and matplotlib to load the data and visualize it. Kaggle data set used -> [Kaggle Student Exam Data](https://www.kaggle.com/datasets/saadaliyaseen/analyzing-student-academic-trends)
